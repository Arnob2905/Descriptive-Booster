📊 Analyzing Income Distribution and Household Demographics
📌 Project Overview
This project analyzes household demographic and income data using statistical techniques and data visualization. The goal is to understand income distribution patterns and demographic relationships using descriptive statistics and visualization tools.

The analysis includes:

Central tendency measures
Measures of dispersion
Distribution analysis
Data visualization
Comparison of income across demographic groups
🎯 Objective
The objective of this project is to explore a dataset containing information about household demographics, income, and education levels.

The analysis focuses on:

Understanding different types of data
Applying statistical concepts
Calculating descriptive statistics
Analyzing distribution properties such as skewness and kurtosis
📂 Dataset Structure
The dataset contains 150+ household records with the following columns:

Column Name	Data Type	Description
Household_ID	Categorical	Unique household identifier
Age_of_Household_Head	Numerical	Age of the household head
Household_Income	Numerical	Monthly household income
Education_Level	Categorical	Primary / Secondary / Graduate / Post-Graduate
Family_Size	Numerical	Number of family members
Owns_House	Categorical	Yes / No
Urban_Rural	Categorical	Urban or Rural location

📌 Key Insights
Income distribution is positively skewed, indicating that a small number of households earn significantly higher incomes.
Higher education levels tend to correlate with higher household income.
Urban households generally earn more than rural households.
The dataset shows income variability, indicating economic inequality among households.
